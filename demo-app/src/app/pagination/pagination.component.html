Pagition

Basic pagination

mz-pagination has property itemsPerPage and totalItems to display the number of page.

Active page

You can add currentPage property to select a specific page when rendering.

Maximum page buttons

If you want to change the number displayed page component you can use maxPageButton property.

Subscribe on page change

You can use the ouput property changePageEvent to call a function when page changed.

HTML Structure
<mz-pagination> (changePageEvent)="function($event)" [currentPage]="1" [itemsPerPage]="10" [totalItems]="100" > </mz-pagination>
