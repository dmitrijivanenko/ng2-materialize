Chip input

Type your tag text and press enter to add tags. You can delete them by either clicking on the close icon or using your delete button.

Chip input initial value

mz-chip initial value can be set using ngModel or when bind to a formControl.

Chip input placeholders

You can specify placeholders by using placeholder input property and/or secondaryPlaceholder property for secondary placeholder when adding additional chips.

Chip input autocomplete

You can configure autocomplete with autocompleteOptions input properties.

Chip input events

mz-chip-input exposes add, delete and select output events for hooking into chips functionality.

HTML Structure
mz-chip

<mz-chip [close]="true" (delete)="onDelete($event)"> <img src="assets/select-icon-1.jpg"> John Doe </mz-chip>

mz-chip-input

<mz-chip-input [autocompleteOptions]="autocompleteOptions" [placeholder]="'Enter a tag'" [secondaryPlaceholder]="'+Tag'" (add)="onAdd($event)" (delete)="onDelete($event)" (select)="onSelect($event)"> </mz-chip-input>
